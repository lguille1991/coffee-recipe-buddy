# 1Zpresso Q-Air Grind Table

Source model: **1Zpresso Q Air** (Honest Coffee Guide)

- Notation: `R.C.M` (rotations.clicks.micro-adjustments), e.g. `2.5.0`

## Recommended Settings by Brew Method

| Brew method | Settings |
|---|---|
| Turkish | 0.1.1 – 0.6.1 |
| Espresso | 0.5.1 – 1.1.0 |
| V60 | 1.2.0 – 2.0.1 |
| Aeropress | 0.9.2 – 2.8.0 |
| Moka Pot | 1.0.2 – 1.9.1 |
| Pour Over | 1.2.1 – 2.7.1 |
| Siphon | 1.1.1 – 2.3.1 |
| Filter Coffee Machine | 0.9.0 – 2.6.1 |
| French Press | 2.0.1 – 3.8.0 |
| Cupping | 1.3.2 – 2.5.0 |
| Cold Brew | 2.3.2 – 4.0.0 |
| Cold Drip | 2.4.1 – 3.7.1 |
| Steep-and-release | 1.3.1 – 2.4.0 |
