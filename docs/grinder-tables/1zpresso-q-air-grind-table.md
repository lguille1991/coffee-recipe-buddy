# 1Zpresso Q-Air Grind Table

The Q-Air uses a 0–10 numerical scale with decimal precision. It is a lighter, travel-friendly grinder.

**Scale:** 0 (finest) to 10 (coarsest). Half-clicks = 0.5 increments.
**Resolution:** ~80–100 microns per whole number.

## Notes

- Report Q-Air settings to one decimal place (e.g., 6.5, not 6).
- Range should span ~0.4–0.6 Q-Air units, matching the K-Ultra click range proportionally.
- Q-Air is a good travel substitute; recommend same starting point process as K-Ultra.
