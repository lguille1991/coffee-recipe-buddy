# 1Zpresso K-Ultra Grind Table

The K-Ultra is the primary reference grinder. All range system click values refer to this grinder.

**Scale:** External adjustment, clicks numbered from 0 (closed/finest) upward.
**Resolution:** ~11 microns per click (approximate).
**Full range for coffee:** ~40–120 clicks.

## Notes

- All values are approximate. Individual units may vary ±5%.
- Offset-adjusted final ranges can go outside these typical zones — that is expected.
- For espresso use, refer to the 40–65 click range (not part of this app).
