# 1Zpresso K-Ultra Grind Table

Source model: **1Zpresso K-Ultra** (Honest Coffee Guide)

- Notation: `R.N.T` (rotation.number.tick), e.g. `0.8.2`
- App rule: store `range_logic.final_operating_range` in internal click range for deterministic math; render K-Ultra `starting_point` in `R.N.T`.

| Brew method | Rotations from zero |
|---|---|
| Turkish | 0.0.6 – 0.2.8 |
| Espresso | 0.2.4 – 0.5 |
| V60 | 0.5.3 – 0.9.2 |
| Aeropress | 0.4.3 – 1 |
| Moka Pot | 0.4.8 – 0.8.6 |
| Pour Over | 0.5.4 – 1 |
| Siphon | 0.5 – 1 |
| Filter Coffee Machine | 0.4 – 1 |
| French Press | 0.9.1 – 1 |
| Cupping | 0.6.1 – 1 |
| Cold Brew | 1 |
| Cold Drip | 1 |
| Steep-and-release | 0.6 – 1 |
